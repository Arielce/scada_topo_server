topo_model_rtdb.xml文件用来描述关系模型
tag:ems表示应用类型
tag:breaker表示设备类型，如开关
tag:tab表示表号
tag:field表示域信息

关系模型转换成图模型的规则:
关系模型描述文件中域信息有link属性，且属性值为1，表示该域为一个连接点，则在图模型中创建一个“点”，点的id即为该域的值；如果模型有两个link属性值为1的域，则将两个点连成一条边，这条边的id为该模型的id；
暂时不支持超过两个link属性值为1的模型。
即：只有两个连接点的模型转换成边，只有一个连接点的模型转换成点。
针对交流线段处理的特殊规则：
交流线段端点转换成“点”，将parent属性值相同的两个点连接起来形成“边”。边的id为其中一个端点的id，边的标签为父对象的类型。
针对变压器处理的特殊规则：
变压器绕组转换成“点”，将parent属性值相同的点连接起来，三绕组变会有三个点，连接起来形成一个三角形，三条边的id分别为一端点的id
注意：配置文件中父设备必须写在子设备之前
